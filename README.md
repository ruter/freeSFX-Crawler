# freeSFX-Crawler
You need import library `requests`, use `pip` to install it:

```
sudo pip install requests
```

Before use, edit `main.py`, to fill your freeSFX account at 'xxxxx':

```
form_data = {
            'username': 'xxxxx',
            'password': 'xxxxx',
            'login': 'Log in'
        }
```
